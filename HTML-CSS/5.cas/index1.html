Kaskadni stilovi

Postoje 3 vrste primjene kaskadnih stilova na html dokumentima

  1. Direktna primjena stilova kao atribut nekog taga koji zelimo da mijenjamo
  2. Unutar head taga imamo style tag unutar kog vrsimo primenu CSS_a
  3. Eksterni CSS.Njega zapravo karakterise novi fajl(.CSS) koji ce biti povezan sa html fajlom
